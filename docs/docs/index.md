# Helena App Docs

Temos que o helena app é um projeto de nascimento acadêmico que visa trazer conceitos presentes dentro do UX e UI design, 
para criar aplicações móveis que possibilitem de maneira mais prática e rápida, o desenvolvimento, perpetuamento, e estabelecimento de novos hábitos, 
de forma a modificar a rotina dos seus usuários trazendo consequências possitivas dos hábitos que foram desenvolvimentos em conjunto com o app. Isso se daria por meio de uma gamificação visual do processo, que seria estimulante para o usuário, de forma que ele conseguisse ver todo o seu progresso de forma facilitada, e 
mais direta, facilitando assim com que o mesmo permanecesse motivado por mais tempo e com mais consistência


Temos que o projeto foi desenvolvido em Flutter dentro da sua aplicação mobile, em React dentro da sua versão web, e que seu back-end é sustentado pelo 
FastAPI em conjunto com outras bibliotecas, como o numpy e o SQLAlchemy, sejam eles para cálculo de métricas ou para o gerenciamento e manipulação de dados 
respectivamente.
